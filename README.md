# Insurance Fraud Analysis Dashboard (Power BI)

## Aim
To analyze insurance claims data and detect patterns of fraudulent activities using Power BI.

## Summary
- Developed an **interactive Power BI dashboard** with **KPI cards, stacked/clustered charts, Waterfall, Treemap, and Trend Line/Area charts**.
- Performed **trend and pattern analysis** over Month-Year, Claim Type, Vehicle/Auto Make, and Incident Hour.
- Added **slicers for Age Group, Incident Type, Policy State, and Month-Year** for dynamic interactivity.

## Pages
1. **Dashboard Overview** – Title page introducing the project
2. **Fraud Insights Summary** – High-level KPIs, Fraud by Incident Type, Age Group, and Policy State
3. **Detailed Analysis & Trends** – Claim Type Waterfall, Vehicle Treemap, Trend Line/Area chart, and Incident Hour analysis

## Result / Insights
- Identified **high-risk incidents, age groups, and policy states** prone to fraud
- Visualized **claim type contributions** and patterns by vehicle and incident hour
- Created a **fully interactive dashboard** for detailed insights and decision-making
